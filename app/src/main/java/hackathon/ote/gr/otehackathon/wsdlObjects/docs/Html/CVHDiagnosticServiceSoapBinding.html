CVHDiagnosticServiceSoapBinding

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
CVHDiagnosticServiceSoapBinding service = new CVHDiagnosticServiceSoapBinding(null,"https://solvrs1.central-domain.root.ote.gr/helpdesk/webservice/diag",30000);
service.getTopology();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
CVHDiagnosticServiceSoapBinding service = new CVHDiagnosticServiceSoapBinding(null,"https://solvrs1.central-domain.root.ote.gr/helpdesk/webservice/diag");
service.getTopology();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
CVHDiagnosticServiceSoapBinding service = new CVHDiagnosticServiceSoapBinding();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.getTopology();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
CVHDiagnosticServiceSoapBinding service = new CVHDiagnosticServiceSoapBinding();
service.enableLogging=true;
service.getTopology();
